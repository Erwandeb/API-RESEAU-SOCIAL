## Pré-Requis :
Pour faire tourner ce projet, vous aurez besoin de NodeJS et de télécharger ce projet.


## Une fois téléchargé :

- Ouvrir le terminal de commande de votre IDE
- NPM Install
- NPM start 
- Le projet doit se lancer sur votre serveur local:3000
